# todo-list-js

A simple To-Do List application built using **HTML, CSS, and JavaScript**.

## Features
- Add tasks dynamically
- Delete tasks
- Mark tasks as completed (toggle)
- Interactive UI with hover effects

## How to Use
1. Open `index.html` in your browser.
2. Type your task in the input box.
3. Click **Add** button (or press Enter if implemented).
4. Click on a task to mark it completed.
5. Click the **X** button to delete a task.

## Technologies Used
- HTML
- CSS
- JavaScript (DOM manipulation)


## Note
- This project is currently **optimized for desktop screens**.  
- On smaller screens (mobile phones or tablets), the layout may not display perfectly.
- They are not stored.(when page refresh t starts with new).
